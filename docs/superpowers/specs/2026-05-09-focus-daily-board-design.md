# Focus 日期看板设计

## 目标

在现有 Focus 模块中增加页内“日期看板”，用于按日期汇总当天计划、完成和未完成任务，并一键复制到日报。

## 入口

- 在 `focus.vue` 主区域顶部增加“日期看板”入口。
- 入口与现有“报表”“设置”并列。
- 打开后主区域切换到日期看板；左侧清单导航保留。
- 支持返回当前任务列表。

## 范围视图

- 最近 7 天：适合补日报。
- 本周：适合周内复盘。
- 月历：适合回看指定日期。

每个日期显示计划数、完成数、顺延数。点击日期后展示该日详情。

## 日报口径

- 今日计划：`dueDate` 等于该日期的任务。
- 今日完成：`completedAt` 落在该日期内的任务。
- 未完成/顺延：`dueDate` 等于该日期且当前未完成的任务。

## 复制格式

```text
今日计划：
- 任务 A

今日完成：
- 任务 A

未完成/顺延：
- 任务 B
```

空段落输出 `- 无`。

## 快照

- 默认实时生成日报内容。
- 用户点击“保存快照”后，将三段任务和复制文本保存。
- 已保存日期优先显示快照。
- 用户可点击“更新快照”覆盖快照。

## 实现边界

- 复用现有 Focus 任务、清单和完成时间数据。
- 新增日报快照存储，不做自动每日冻结。
- 第一版不追踪任务历史变更。
- 不改变现有任务列表、番茄钟和报表行为。

## 验证

- 能在三种范围视图间切换。
- 任意日期能正确展示三段内容。
- 复制文本格式稳定。
- 保存和更新快照后刷新页面仍可读取。
- 现有 Focus 任务增删改和番茄钟流程不回归。
