# Agent Notes

- 本项目的 Web 开发服务必须固定启动在 `http://127.0.0.1:3000/`。
- 启动前如果 `3000` 端口已被占用，先确认占用进程并杀掉它，再重新启动本项目；不要自动改用 `3001`、`3002`、`3003` 等其它端口。
- 如果需要启动 Nuxt，使用 `web-ui` 目录下的 `npm run dev -- --host 127.0.0.1 --port 3000`。
- 每次修改本项目文件后，主动将本次修改 commit 到本地 Git；提交时只包含当前任务相关文件，不混入无关改动。
- 不要在 TS/JS 里用多行字符串拼复杂脚本逻辑，尤其不要把 PowerShell/Bash 函数内联进命令参数；复杂更新、同步、清理逻辑应放到独立脚本文件里，TS/JS 只负责参数校验、编排和调用。
